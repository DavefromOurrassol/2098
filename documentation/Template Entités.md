
**Définition : une instance = incarnation d’une entité dans un scénario donné, à un moment donné, avec une fonction propre**

---

# 🟦 IDENTITÉ

- **Nom instance** _(Title)_
- **Entité liée** _(Relation → ENTITÉS)_
- **Scénario** _(Relation → SCÉNARIOS)_

---

# 🧠 NATURE CONTEXTUELLE

- **Type dans scénario** _(Select)_  
    IA / organisation / entreprise / institution / infrastructure / réseau / humain / système / hybride / autre
- **Rôle dans scénario** _(Long text)_  
    → fonction narrative + systémique
- **Responsabilités** _(Long text)_  
    → actions concrètes exercées dans le monde

---

# ⚙️ IMPACT SYSTÉMIQUE

- **Impact local (0–5)** _(Number)_
- **Impact systémique global (0–5)** _(Number)_

---

# ⚙️ VARIABLES & SYSTÈME

- **Variables influencées** _(Relation → VARIABLES)_  
    → effets causaux directs

---

# 🌍 ZONES

- **Zone géographique** _(Multi-select)_  
    locale / urbaine / nationale / régionale / continentale / globale
- **Zone systémique** _(Multi-select)_  
    énergie / IA / gouvernance / économie / information / sécurité / infrastructure / société / cyberspace / orbital

---

# 🤝 RELATIONS ENTRE INSTANCES

- **Alliances** _(Relation → INSTANCES)_
- **Oppositions** _(Relation → INSTANCES)_
- **Type de relation dominante** _(Select)_  
    coopération / alliance stratégique / dépendance / neutralité / rivalité / conflit / infiltration / symbiose

---

# ⏳ TEMPS

- **Année début** _(Number)_
- **Année fin** _(Number ou vide)_
- **État temporel** _(Select)_  
    actif / disparu / transformé / clandestin / historique / mythifié
- **Âge historique** _(Select)_  
    émergent / marginal / ascendant / dominant / mature / déclinant / résiduel / mythifié
- **Génération** _(Select)_  
    pré-crise / transition / post-effondrement / IA-native / forteresse / reconstruction / ère cognitive

---

# 🎭 NARRATIF

- **Description journalistique** _(Long text)_  
    → version exploitable pour articles
- **Signes distinctifs** _(Long text)_  
    → éléments visuels / symboliques
- **Tensions narratives** _(Long text)_  
    → conflits, enjeux, trajectoires possibles
