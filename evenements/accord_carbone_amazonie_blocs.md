# Traité de Belém — Souveraineté Carbone Amazonienne

## Description archétypale
Signature a Belem du premier traite de souverainete carbone entre le bassin amazonien et les trois grands blocs geopolitiques. Le Consortium Amazonia Viva impose des conditions sans precedent : veto sur les projets industriels dans un rayon de 500km, redevance carbone directe aux peuples autochtones. Certains blocs refusent de signer, creant une fracture diplomatique majeure.
