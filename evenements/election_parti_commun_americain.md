# Élection du Parti Commun Américain

## Description archétypale
Un parti se réclamant du communisme remporte les élections présidentielles
américaines en 2055, première victoire de ce type depuis la fondation des
États-Unis. Événement structurant dont la réalité, l'intensité et les
conséquences varient radicalement selon les mondes.
