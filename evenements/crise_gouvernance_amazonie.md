# La Fracture de Belém — Quand le Bassin Amazonien a cessé d'être un territoire

## Description archétypale
Le Consortium Amazônia Viva déclare un état d'urgence écologique sur 40% du bassin amazonien après la détection de déforestations illégales massives orchestrées par des opérateurs privés liés aux blocs énergétiques. Belém devient le théâtre d'une confrontation entre les gardiens autochtones et les milices corporatives. L'ONU est saisie mais incapable d'intervenir. Premier test de souveraineté pour le bassin amazonien comme acteur géopolitique autonome.
