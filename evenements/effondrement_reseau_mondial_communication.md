# Effondrement du Réseau Mondial de Communication

## Description archétypale
Un attentat coordonné détruit simultanément 9 nœuds critiques du réseau
mondial de communication en 2040, plongeant 3,2 milliards de personnes
dans une coupure totale pendant 17 à 90 jours selon les régions.
Événement fondateur dont les conséquences sur l'architecture numérique
mondiale sont irréversibles.
